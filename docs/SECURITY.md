# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.0-beta | ✅ |

## Reporting a Vulnerability

Email: kauenet@superteam.fun
Disclosure policy: 90-day responsible disclosure.
We will acknowledge within 48 hours and provide a fix timeline within 7 days.

## Known Limitations

- Single keypair authority in current prototype. Production upgrade path: Squads v4 multisig (SQDS4ep65T869zMMBKyuUq6aD6EgTu8psMjkvj52pCf). See ARCHITECTURE.md for the full upgrade path.
- Devnet-only deployment. Not audited for mainnet use.

## Out of Scope

- Devnet-only deployments
- Issues requiring physical access to keypair files
- Bugs in dependencies (report directly to upstream)
